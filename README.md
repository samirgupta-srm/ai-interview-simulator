# AI Interview Simulator

A simple AI-powered interview simulation platform.

## Features
- Generate interview questions based on job role
- Candidate answer input
- Interview score generation

## Tech Stack
Frontend: HTML, JavaScript
Backend: Node.js, Express

## How to Run

1 Install Node.js

2 Install dependencies

npm install

3 Start the server

node server.js

4 Open browser

http://localhost:3000

## Project Purpose

This project was built for the Hack & Forge Hackathon to demonstrate an intelligent interview simulation platform.
